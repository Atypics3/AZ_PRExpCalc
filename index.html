Total EXP Calculator

The way this calculator works is as follows:

  • Select the current phase and the appropriate type for the ship that you're working on.
    • Phase One means that 1,000,000 EXP needs to be collected in order to move on.
    • Phase Two means that 2,000,000 EXP needs to be collected in order to move on.
  • Select the amount of ships that are going to accumulate EXP; either Vanguard or Backline.
  • Enter the current and target percentages of your Priority/Decisive ship.
  • For the EXP Mode Select section, there are two modes that can be chosen:
    • Random Mode:

      • The nodes fought for each run will be randomized six times with the exception of the boss node, which is counted only once.

      For more information for nodes, click here.

    • Optimal Mode:
        • The enemy nodes that are battled are as follows:
      • - 3 small fleet enemy nodes
      • - 2 medium fleet enemy nodes
      • - 1 heavy fleet enemy node
      • - 1 boss node

  • The ships used will always have Good morale or better throughout the entire duration of the run.
    In most cases, this will almost always be a constant unless in the case of heavy grinding. In which case, the runs needed would be higher.
  • The battle rank will always be S-rank.
  • There are not any exp-boosting ships present in any of the fleets.
  • The Runs Needed section calculates to the nearest tenth place.
